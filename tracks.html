Track 1: Open Track

Motivation

The Open Track is the primary competition track. It is designed for maximum flexibility: use any model, any provider, and any architecture to build the most reliable agent possible. The goal is to close the gap between frontier model capability and deployment-ready consistency on CAR-bench.

Current state-of-the-art achieves only 58% consistent success (Pass^3). There is significant room for improvement through better scaffolding, prompting, planning, and agent design—independent of the underlying model.

What You’ll Build

A dockerized agent that communicates via the A2A (Agent-to-Agent) protocol with the CAR-bench evaluation environment. Your agent receives user requests, interacts with tools, and produces responses that are evaluated for correctness, policy compliance, limit-awareness, and disambiguation behavior.

Allowed Approaches

  • Prompting: System prompts, few-shot, chain-of-thought, self-reflection
  • Scaffolding: Planning-execution separation, tool-use strategies, retry logic
  • Multi-agent: Subagents, agent skills, orchestration patterns
  • Fine-tuning: SFT, RLHF, DPO on the training data
  • Context engineering: RAG over environment data, dynamic context management
  • Any combination of the above

Evaluation

Teams are ranked by Pass^3 on the hidden test set (125 tasks). A task scores 1 only if solved correctly in all 3 independent trials. Per-category breakdown (Base / Hallucination / Disambiguation) is shown alongside the aggregate score.

Prizes

  • Rank Awards: 1st, 2nd, 3rd place by Pass^3 on hidden test set
  • Best Innovation Award: For novel approaches — smaller models, creative architectures, efficiency-focused solutions
  • Certificates for award winners; top teams present at the IJCAI competition session

Track 2: Cerebras Fast-Reasoning Track

Motivation

Cerebras builds the world’s fastest AI inference infrastructure, holding independently verified speed records on frontier-scale models. This track explores what becomes possible when inference latency is no longer a bottleneck: can agents use deeper reasoning, self-reflection, or search strategies that would be infeasible under conventional time constraints?

While the Open Track rewards raw performance, this track rewards innovative use of compute headroom. The question is not just "how well does your agent perform?" but "what does your agent do with the extra inference budget that wouldn’t be possible otherwise?"

What You’ll Build

Same A2A-compatible agent format as Track 1, but your agent runs on Cerebras inference infrastructure within a fixed time budget. The additional speed budget is the key differentiator—use it for multi-pass reasoning, Monte Carlo tree search, generate-and-verify, or any strategy that benefits from more inference calls.

Allowed Approaches

  • All approaches from Track 1 are permitted
  • Additionally encouraged: Multi-pass reasoning, self-reflection loops, best-of-N sampling, search-based planning, verification chains, ensemble approaches
  • Agents must run on Cerebras-provided infrastructure

Evaluation

Ranked by a combination of innovative use of compute headroom and Pass^3 performance. Judges evaluate whether the approach meaningfully leverages fast inference in ways that differ from standard single-pass agent design.

Prizes

  • Cerebras Innovation Award: For the best demonstration of compute-aware agent design
  • Cerebras-sponsored inference credits for development (bound to quota)
  • Certificates and presentation slot at the IJCAI competition session
